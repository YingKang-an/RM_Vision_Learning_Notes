我隶属于沈阳航空航天大学T-up战队视觉组,本仓库是我(YinKang'an)的学习任务笔记仓库 ,记录我的学习任务实践过程与知识点总结,是跟随战队学习路线,逐步完成的阶段性学习成果汇总

## 📖 项目介绍
本仓库是沈阳航空航天大学T-up战队视觉组**个人学习任务归档仓库**,核心用于记录和整理我跟随战队「视觉组学习路线」完成的各项学习任务,代码实践及知识点总结.

仓库采用「任务模块分目录」的管理方式,每个任务对应独立的代码实现,配置文件及学习笔记 ,既能用于我个人复盘学习过程,梳理技术难点,也可作为战队视觉组新生入门学习的参考示例素材

## ⚙️ 环境要求
- 操作系统:Linux (推荐 Ubuntu 24.04+)
- 编译工具:GCC/G++ ≥ 9.4.0、CMake ≥ 3.16 
- 依赖库(按需安装)
  - OpenCV（≥ 4.6.0）: 核心图像处理工具
  - yaml-cpp : YAML 配置文件解析工具
  - ROS2 Humble/Jazzy: 机器人开发框架

## 🎯 仓库用途
1.  个人学习复盘:详细记录每个学习任务的实现流程,踩坑经验与知识点总结 
2.  **战队经验传承:为T-up战队视觉组后续新生,提供可参考的学习示例与思路** 
3.  任务进度跟踪:按战队学习路线归档各阶段任务,清晰展示个人学习进度与成长 

## 📢 备注
- 仓库持续更新,跟随战队学习路线完成新任务后,会同步归档对应的代码与笔记
- 如有任务相关技术问题,优先通过战队内部交流渠道沟通探讨
