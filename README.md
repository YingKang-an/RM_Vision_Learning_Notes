# 🚀 RM_Vision_Learning_Notes
沈阳航空航天大学T-up战队视觉组学习任务笔记仓库📚，记录视觉组新生从基础到进阶的学习任务实践过程与知识点总结，是跟随战队学习路线完成的阶段性学习成果汇总✨。

## 📖 项目介绍
本仓库是沈阳航空航天大学T-up战队视觉组的个人学习任务归档仓库🗂️，用于记录和整理跟随战队「视觉组学习路线」完成的各项学习任务、代码实践、知识点总结。仓库按任务模块分目录管理，每个任务对应独立的代码实现、配置文件及学习笔记📝，既用于个人复盘学习过程，也可作为战队视觉组新生学习参考的示例素材💡。

## ⚙️ 环境要求
- 操作系统：Linux（推荐Ubuntu 20.04+/22.04+，适配Robomaster开发环境🖥️）
- 编译工具：GCC/G++ ≥ 9.4.0、CMake ≥ 3.16🛠️
- 依赖库（按需安装）：
  - OpenCV（≥ 4.5，图像处理📷）
  - yaml-cpp（YAML解析🔧）
  - ROS2 Humble/Jazzy（机器人框架🤖）

## 🎯 仓库用途
1. 个人学习复盘：记录每个学习任务的实现过程、踩坑经验与知识点总结📌；
2. 战队经验传承：为T-up战队视觉组后续新生提供学习参考示例👨‍🎓；
3. 任务进度跟踪：按战队学习路线归档各阶段任务，清晰展示学习进度📈。

## 📢 备注
- 各任务目录下会补充该任务的具体实现细节、知识点笔记📋；
- 仓库持续更新🔄，跟随战队学习路线完成新任务后会同步归档对应的代码与笔记；
- 如有任务相关问题，可结合战队内部交流渠道沟通💬。
