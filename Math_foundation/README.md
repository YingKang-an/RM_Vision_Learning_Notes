## 📚数学基础库 - MATH_FOUNDATION

### 🎯**项目目标**
这个数学基础库旨在为编程项目提供坚实的数学理论基础支持!💪

### 📖**库内容概览**

#### 🔢 **1. 线性代数基础** 🧮
- **矩阵运算**：矩阵乘法、逆矩阵、特征值分解等
- **向量空间**：向量运算、线性变换、基变换
- **坐标系转换**：世界坐标系、相机坐标系、像素坐标系转换

#### 📈 **2. 优化算法** ⚡
- **梯度下降算法家族**：
  - 🎯 随机梯度下降 (SGD)
  - 📦 批量梯度下降 (BGD)  
  - 🔄 小批量梯度下降 (Mini-batch GD)
- **收敛性分析**：学习率选择、收敛速度证明

#### 📊 **3. 概率统计** 🎲
- **概率分布**：正态分布、均匀分布等
- **统计推断**：参数估计、假设检验
- **贝叶斯理论**：先验概率、后验概率推导

#### 🎨 **4. 计算机视觉数学** 👁️
- **相机几何**：透视投影、相机标定
- **图像变换**：仿射变换、透视变换
- **特征提取**：SIFT、HOG等算法的数学原理
